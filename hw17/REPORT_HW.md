# Ознакомление с PyLint

## Порядок установки:
- В терминале с помощью команды __pip install pylint__ установим данную программу.
- Перезагружаем IDE

## Использование
В терминале найдем файл, который хотим проверить и напишем команду __pylint <file_name.py>__
